# Cubo 3D com Three.js

Este projeto apresenta uma cena 3D desenvolvida com Three.js, contendo um cubo visível em uma sala, posicionado sobre uma mesa.

A aplicação utiliza uma câmera em perspectiva, permitindo visualizar o objeto com profundidade. Também possui interação com o mouse, possibilitando rotação, zoom e movimentação da câmera.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Three.js
- Vite

## Funcionalidades

- Exibição de um cubo 3D
- Cena 3D com mesa, sala, paredes, piso e iluminação
- Câmera em perspectiva
- Interação com mouse usando OrbitControls
- Rotação, zoom e movimentação da câmera
- Ajuste automático ao redimensionar a janela
